# Changelog

All notable changes to this project will be documented in this file.

The format is inspired by Keep a Changelog and follows Semantic Versioning.

---

## [v0.3.0] - 2026-07-06

### Added
- Native VaultModule contract
- Platform Event Bus
- Dynamic module metadata
- Module lifecycle management
- Native Document Vault integration
- Event-driven platform communication
- Dashboard health and statistics

### Changed
- Document Vault now operates as a native platform module
- Module Manager supports VaultModule lifecycle
- Platform dashboard displays live module information

---

## [v0.2.0] - 2026-07-05

### Added
- Platform authentication
- Session Manager
- Activity Monitor
- Auto-lock
- Platform Settings
- Security Center
- Shared authentication through VaultCore

### Changed
- Authentication moved from Document Vault to VaultCore
- Single platform login introduced

---

## [v0.1.0] - 2026-07-05

### Added
- VaultCore foundation
- Platform dashboard
- Module Manager
- Logger service
- Notification service
- Theme service
- Shared infrastructure
- Document Vault registered as Module 1
